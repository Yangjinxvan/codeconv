# Windows1252和ASCIII转换器
## 1. 功能说明
- 核心作用：把字符转换成对应的编码，或把编码转换回对应的字符
- 适用场景：需快速转换单字节编码的场景

## 2. 运行环境
- 依赖：Python 3.10+
- 所需库：
	1. 必装
		1. keyboard（实现快捷键）
		2. loguru（用来输出日志）
		3. pywin32（仅Windows系统装）
	2. 选装
		1. tqdm（好看的进度条，推荐装），下载方法：
	    ```bash
      pip.exe install E:/离线安装库/tqdm.whl
		  ```

## 3. 使用步骤
1. 用管理员模式打开终端
2. 输入：
  ```bash
  python codeconv.py
  ```
3. 启动后可输入请“帮助”查看使用方法

## 4. 注意事项
1. 运行前请确保U盘未被设为「只读」模式
2. 若报错“找不到库”，复制后面的库名，在终端输入：
  ```bash
  pip install 库名.whl
  ```
  把库名替换成你刚才复制的内容
